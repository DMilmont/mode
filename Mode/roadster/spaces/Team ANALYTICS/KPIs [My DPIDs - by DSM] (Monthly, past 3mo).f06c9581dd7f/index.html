Caution:

  • The 'Visitor' counts in this report will be slightly higher than the Visitor Counts you see in Dealer Admin. The reason is that we wanted to make this report run fast - and took a shortcut. The Dealer Admin reports only look at Unique Visitors during the entire month. If someone visited 3 times on Monday and 2 times on Tuesday, the dealer admin reports would count this person as '1 Visitor'. The report below only de-dupes at the day level so for that same example, that person would count as '2 Visitors' - one visit on Monday, one visit on Tuesday. It doesn't invalidate this report - you can still compare one DPID to another. Just realize you shouldn't quote these numbers to dealers. Always use the Dealer Group Reports for that.
