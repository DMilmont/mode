FAQ:

Q: My customer's DPID is not in the dropdown

A: That means no CDK data is flowing. Check this report to see if they have CDK ID and/or sales match data. If you see sales match data but no CDK ID, they might be getting Dealer Socket data - there's a 2nd report for Dealer Socket DPIDs. If they do have a CDK ID and but now sales match data then CDK data isn't flowing - work with support to figure out why.
